Voice & Tagline

Primary Tagline

Educate. Entertain. Inspire.

Usage Rules

  • Always three words, each ending with a period. Never truncate.
  • All caps when set in Oswald/display type.
  • Eyebrow labels: VPM News — yellow, uppercase, letter-spacing 0.12em.
  • Brand voice: direct, local, public service. Avoid corporate jargon.
  • CTAs: strong imperatives — Keep Me Informed, Watch Live, Read More.

Accent Bar

3-color gradient stripe

A 5px horizontal stripe placed at the very top of pages and components. Runs left-to-right: Light Blue → Yellow → Red in equal thirds. Never scale to taller heights — it reads as decoration, not a banner.

/* Accent bar — top of every page/component */
.accent-bar {
  height: 5px;
  background: var(--accent-bar);
}

/* Token definition */
--accent-bar: linear-gradient(90deg,
  var(--vpm-light-blue) 0%,   var(--vpm-light-blue) 30%,
  var(--vpm-yellow)     30%,  var(--vpm-yellow)     70%,
  var(--vpm-red)        70%,  var(--vpm-red)        100%);

Component Conventions

Rules every widget must follow
  • Namespace All class names and IDs prefixed with a widget-specific slug. Example: vpm-elec26-, vpm-mm-. Prevents style collisions in Brightspot/WordPress host pages.
  • Naming BEM convention within the namespace: vpm-widget__element--modifier. Keeps specificity flat and predictable.
  • Encapsulation Use Shadow DOM for reusable web components. For simple embeds, scoped namespacing is sufficient.
  • JavaScript IIFE-wrapped. No document.write(). No global variable pollution. Vanilla JS only — no jQuery, React, or Vue unless the host page already provides it.
  • Self-Contained Each widget must work dropped as a single file (or ACF split set) into any CMS. No external CSS framework dependencies. All styles inline or scoped.
  • Accessibility WCAG 2.1 AA minimum. Semantic HTML, keyboard-navigable interactive elements, aria-label on icon-only controls, :focus-visible styles required.
  • Color on dark On dark-blue backgrounds, body text is white. Secondary text is rgba(255,255,255,0.6). Yellow (--vpm-yellow) is used for eyebrows and interactive accents — never as body text on white.
  • Reduced Motion Always include a @media (prefers-reduced-motion: reduce) block that disables transitions and hover transforms.
  • File Format Self-contained: index.html (all CSS/JS inline). ACF split-file: html.html, css.css, js.js, preview.html. Always include a README.md.
