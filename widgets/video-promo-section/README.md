# Video Promo Section

Two-column embed: video player (left) with title, description, and CTA links (right). Poster image with click-to-load Vimeo player. Supports a LIVE bug overlay.

## Customizing

| Attribute / element | What to change |
|---|---|
| `data-vimeo-src` on the poster button | Vimeo embed URL |
| `<img src>` inside the poster | Poster/thumbnail image |
| `aria-label` on the poster button | Accessible video title |
| `.vpm-ls__livebug` | Remove this element if not a live event |
| `.vpm-ls__kicker` | Short label above the title (e.g. "Live Coverage", "Watch Now") |
| `.vpm-ls__title` | Main headline |
| `.vpm-ls__desc` | Supporting copy |
| `[data-vpm-play]` button text | Primary CTA label |
| `.vpm-ls__btn--secondary` href + text | Secondary link |
